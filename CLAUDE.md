# CLAUDE.md

## Overview

This repository contains original templates, skills, agents, and utilities for AI compliance workflows in automotive settings.

## Architecture

- `agents/`: regulatory and safety subagents
- `skills/`: workflow guidance
- `templates/`: artifact drafts
- `src/`: traceability helpers
- `tests/`: light regression tests

## Quality standards

Outputs should be:
- auditable
- explicit about confidence
- clear about review ownership
- separated into requirements, controls, evidence, and gaps
